Sobre Mim

Olá! Meu nome é Gabriel e tenho 18 anos. Iniciei meus estudos intensivos na área de tecnologia há 2 anos, focando na programação. Atualmente, sou estudante de Ciência da Computação (CEUB), com formação complementar em Programação Web (SENAC). Tenho focado no desenvolvimento web full-stack dinâmico, com ênfase no domínio de JavaScript e PHP, e possuo uma base sólida em SQL para gerenciamento de dados em meus projetos autorais. Este repertório técnico se expande para o desenvolvimento com Python e noções de Java, garantindo uma visão abrangente e adaptável às diferentes demandas do desenvolvimento de software.
